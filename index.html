Skype Historian

save your chats easily

About

When you reinstall your operating system or change you computer you lose all your chats history. Skype Historian allows you to easily backup all your Skype chats to files in different formats. Files can also be compressed into a single ZIP archive.

Skype Historian allows to export your chats into the following formats:

  • Plain Text Files
  • JavaScript Object Notation
  • Comma-separated values
  • eXtensible Markup Language

System Requirements

Skype Historian runs pretty well on:

  • Windows 8
  • Windows 7
  • Windows XP with .NET Framework 3.5 (or above) installed.
